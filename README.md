# dotfile

My own system setting about **bash**, **neovim editor**, **tmux multiplexer**.

# Installation
Execute following command first, and check if you have all the prequisited program installed.
```
# Clone the repo
git clone https://github.com/johnnylord/dotfile ~/.dotfile

# Execute the install script
~/.dotfile/install.sh
```
> Install all of the dependent program first before you can get into the real meat of the install.sh
